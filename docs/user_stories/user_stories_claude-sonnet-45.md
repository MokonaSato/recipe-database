# ユーザーストーリー

## ユーザーストーリー1: 田中美咲の場合

金曜日の夜、美咲は21時過ぎに帰宅した。プロジェクトの進捗会議が長引き、疲労困憊だ。パートナーは先に夕食を済ませており、美咲はコンビニ弁当を温めて食べた。鏡を見ると、また肌荒れがひどくなっている。外食とコンビニ食続きで、野菜をまともに摂れていない。むくみもひどく、お気に入りのスカートがきつい。「このままじゃまずい」と思いながらも、疲れた頭で「明日は何を作ろう」と考える気力は湧かない。

週末、少し余裕のある朝。美咲はコーヒーを淹れながらTwitterを眺めていると、フォロワーが「献立考えるの面倒すぎる」とつぶやき、誰かが「このアプリ使ってみたら？」とリプライしているのが目に入った。「AI献立提案アプリ」。美咲は興味を持ってダウンロードした。

アプリを開くと、AIが「今週の献立を一緒に考えましょう」と話しかけてくる。美咲は「パートナーが魚介苦手で、辛いのもダメなんです。あと、野菜多めで栄養バランス良くしたい」と入力した。すると、1週間分の献立が提案される。月曜は鶏肉と野菜の蒸し焼き、火曜は豚肉と小松菜の炒め物、水曜は…。「これなら作れそう」と思える、シンプルだけど栄養バランスの良さそうなメニューが並ぶ。

「木曜は外食予定があるから削除」と調整すると、買い物リストが自動生成された。鶏もも肉300g、小松菜1束、しめじ1パック…。美咲はリストを見ながらスーパーへ向かった。いつもなら「何を買えばいいんだっけ」と売り場を何度も往復していたが、今日は迷わない。

月曜の夜、美咲は提案された献立通りに鶏肉と野菜の蒸し焼きを作った。「何作ろう」と悩む時間がないだけで、こんなに気持ちが楽になるとは。パートナーも「美味しい」と喜んでくれた。以前Twitterで見かけて気になっていたレシピをアプリに保存しておくと、来週の献立に組み込まれている。「これなら続けられそう」美咲はそう感じた。

## ユーザーストーリー2: 佐藤健太の場合

水曜日の夕方、健太は18時に仕事を切り上げて保育園に子どもたちを迎えに行った。妻の帰宅は20時過ぎになる。帰宅すると、5歳の息子が「おなかすいた！」と訴える。冷蔵庫を開けるが、何を作ればいいか思いつかない。上の子はピーマンとトマトが嫌い、下の子は肉の脂身が嫌い。妻は減塩・低糖質を希望している。全員が食べられるメニューを考えるのは、子どもの相手をしながらでは至難の業だ。結局、カレーライスという定番に落ち着いた。またカレーか、と自分でもうんざりする。

週末の朝、健太はTwitterで育児アカウントの投稿を見ていた。「家族全員の好みに対応した献立提案アプリ、めっちゃ助かる」というツイート。健太はすぐにダウンロードした。

アプリを開き、家族構成と各自の好みを登録する。「長男5歳、ピーマンとトマト苦手。次男3歳、肉の脂身苦手。妻、減塩・低糖質希望」。AIが「わかりました。家族みんなが食べられる献立を考えますね」と返してくる。

提案された献立を見て、健太は驚いた。月曜は鶏むね肉の照り焼きとほうれん草のおひたし、火曜は豆腐ハンバーグと野菜スープ。ピーマンもトマトも使わず、脂身の少ない肉を使い、減塩でも美味しそうなメニューが並んでいる。栄養バランスも考えられている。「水曜は妻が作るから削除」「金曜のハンバーグは子どもが好きだから残す」と調整した。

土曜の午前中、健太は買い物リストを片手に子どもたちとスーパーへ。「パパ、これ買うの？」と聞かれても、「うん、これでハンバーグ作るんだよ」と答えられる。リストがあるから迷わない。子どもがぐずる前に買い物を終えられた。

月曜の夜、健太は鶏むね肉の照り焼きを作った。子どもたちが「おいしい！」と言ってくれる。妻も「健康的でいいね」と笑顔だ。献立を考える負担が減っただけで、料理が苦痛ではなくなった。家族との食卓が、また楽しくなった気がする。

## ユーザーストーリー3: 山田彩香の場合

火曜日の昼、彩香は在宅ワークの合間にコンビニのおにぎりを頬張った。デザインの締め切りが迫っており、料理をする余裕はない。夕方、ふと鏡を見ると、肌がくすんでいる。「また食事がおろそかになってる」。彩香は自分を責めた。冷蔵庫には先週買った野菜が萎びている。一人分の献立を考えるモチベーションが湧かず、結局使わないまま捨てることになりそうだ。「自分だけのために料理するのって、こんなに面倒なんだ」。

週末、彩香はInstagramの#自炊女子タグを眺めていた。誰かの投稿に「一人暮らしでも献立考えてくれるアプリ見つけた！」とあった。彩香は興味を持ち、アプリをダウンロードした。

アプリを開くと、AIが「今週の食生活、一緒に考えましょう」と優しく話しかけてくる。彩香は「一人暮らしで、今週は仕事が忙しいです。でも野菜はしっかり摂りたい」と入力した。すると、シンプルで栄養バランスの良い献立が提案される。月曜は鮭と野菜のホイル焼き、火曜は豆腐と野菜の味噌汁、水曜は…。一人分でも無駄なく、季節の食材を使ったメニューだ。

「木曜は友人と外食だから削除」「火曜の味噌汁は2食分作って翌日も食べる」と調整した。以前Instagramで保存していた「春キャベツの簡単サラダ」のレシピをアプリに追加すると、献立に組み込まれた。買い物リストには、鮭1切れ、春キャベツ1/4個、豆腐1丁…と、一人分に最適化された量が表示される。

月曜の夜、彩香は鮭と野菜のホイル焼きを作った。調理時間は20分程度。「自分だけのために」という心理的ハードルが、AIに献立を考えてもらうことで下がった気がする。温かい食事を前にして、彩香は「自分を大切にしてる」と感じた。食材も無駄にならず、罪悪感もない。これなら続けられそうだ。
